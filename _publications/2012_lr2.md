Algebraic techniques based on semirings have previously been used to describe shortest path algorithms and routing procedures. These approaches often use matrices over semirings or Kleene algebras, which again form semirings or Kleene algebras. While successful for shortest path algorithms, these approaches fail when modelling timing aspects of routing algorithms, since distributivity laws may no longer hold, which in turn destroys associativity on the matrix level.

This talk presents these shortcomings using the Ad hoc On-Demand Distance Vector (AODV) routing protocol as an example. AODV is a widely used routing protocol for wireless mesh networks and forms the basis of further standards such as IEEE 802.11s. The talk argues that Kleene modules provide a suitable alternative framework. Crucial aspects of routing protocols in general, and AODV in particular, can be modelled naturally using modules. The presentation concludes with a discussion of ongoing developments and future work.
